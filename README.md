# Простой список задач

Общий туду-лист без регистрации. Сервер на Express + SQLite.

## Запуск

```
npm install
npm start
```

## Деплой на Railway
1. Создать новый веб-сервис из этого репозитория.
2. Railway сам установит зависимости и запустит `node app.js`.
3. Healthcheck настроен на `/health`.
4. Приложение будет доступно по выданному домену.
